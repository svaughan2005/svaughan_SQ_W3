# svaughan_SQ_W3

## Setup and Interaction Instructions

To run the sketch, visit it's github page at:

**Player 1 Controls:**

- Move: A / D
- Attack: F
- Block: G

**Player 2 Controls:**

- Move: Arrow Keys
- Attack: K
- Block: L

Press **ENTER** to start or rematch.
